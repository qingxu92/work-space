## 4. 交互设计

### 4.1 全局交互

- **加载状态**：数据请求过程中需展示 Loading 状态。
- **空状态**：无数据时需展示统一的 Empty 状态图。
- **网络异常**：网络断开时提示"网络异常，请检查网络连接"。
- **下拉刷新**：支持下拉刷新更新页面内容。

### 4.2 用户状态交互

| 用户状态 | 交互差异 |
| :----------- | :--------------- |
| 未登录 | 部分功能受限，点击时提示登录 |
| 已登录 | 全部功能可用 |
| VIP用户 | 享受额外权益，如专属标识、折扣等 |

### 4.4 异常处理

| 异常场景 | 处理方式 |
| :----------- | :--------------- |
| 未登录 | 弹出登录弹窗或跳转登录页 |
| 数据加载失败 | 提示"加载失败，点击重试" |
| 操作失败 | 提示失败原因，引导用户重试 |
| 商品下架 | 提示"商品已下架"，显示推荐商品 |
