## 4. 交互设计

### 4.1 全局交互

- **加载状态**：数据请求过程中需展示 Loading 状态。
- **表单验证**：实时验证字段，错误时显示红色边框和错误提示。
- **响应式**：页面需适配不同分辨率屏幕，保证内容不遮挡、不错位。

### 4.2 特殊交互

> 描述页面中特殊的交互逻辑，如：
> - 字段联动（选择A后B字段可选值变化）
> - 条件显示（满足条件才显示某些字段）
> - 异步验证（如用户名重复检查）

### 4.3 异常处理

| 异常场景 | 处理方式 |
| :----------- | :--------------- |
| 必填字段为空 | 提交时校验，提示"请填写XXX" |
| 格式不正确 | 实时校验，提示正确格式 |
| 网络请求失败 | 提示"保存失败，请重试" |
| 数据重复 | 提示"XXX已存在，请修改" |
