# 教育类页面类型

## 页面类型列表

| 序号 | 页面类型 | 英文标识 | 说明 |
|------|---------|---------|------|
| 1 | 首页 | home | 课程入口、学习概览 |
| 2 | 课程列表 | course-list | 课程搜索、分类 |
| 3 | 课程详情 | course-detail | 课程信息、报名入口 |
| 4 | 学习页 | learning | 视频播放、学习进度 |
| 5 | 作业页 | homework | 作业列表、提交 |
| 6 | 个人中心 | user-center | 学习记录、证书 |

## 页面特征

### 课程详情 (course-detail)

| 特征 | 说明 |
|------|------|
| 核心功能 | 展示课程信息、引导报名 |
| 关键模块 | 课程封面、课程介绍、目录、评价、报名按钮 |
| 用户目标 | 了解课程、决定报名 |

### 学习页 (learning)

| 特征 | 说明 |
|------|------|
| 核心功能 | 视频播放、学习进度 |
| 关键模块 | 视频播放器、目录、笔记、讨论 |
| 用户目标 | 完成学习 |

### 作业页 (homework)

| 特征 | 说明 |
|------|------|
| 核心功能 | 作业查看、提交 |
| 关键模块 | 作业列表、作业详情、提交区 |
| 用户目标 | 完成作业 |
