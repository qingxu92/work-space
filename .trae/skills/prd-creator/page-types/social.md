# 社交类页面类型

## 页面类型列表

| 序号 | 页面类型 | 英文标识 | 说明 |
|------|---------|---------|------|
| 1 | 首页 | home | 动态入口 |
| 2 | 动态流 | feed | 关注/推荐动态 |
| 3 | 个人主页 | profile | 个人信息、动态 |
| 4 | 消息页 | message | 私信、通知 |
| 5 | 发布页 | publish | 发布动态 |
| 6 | 设置页 | settings | 账户设置 |

## 页面特征

### 动态流 (feed)

| 特征 | 说明 |
|------|------|
| 核心功能 | 展示动态、互动 |
| 关键模块 | 动态卡片、点赞/评论/转发、刷新 |
| 用户目标 | 浏览动态、互动 |

### 个人主页 (profile)

| 特征 | 说明 |
|------|------|
| 核心功能 | 展示个人信息、动态 |
| 关键模块 | 头像/昵称/简介、关注/粉丝、动态列表 |
| 用户目标 | 了解他人、查看动态 |

### 消息页 (message)

| 特征 | 说明 |
|------|------|
| 核心功能 | 私信、通知 |
| 关键模块 | 会话列表、聊天窗口、通知列表 |
| 用户目标 | 沟通、查看通知 |
