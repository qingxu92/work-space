# 金融类页面类型

## 页面类型列表

| 序号 | 页面类型 | 英文标识 | 说明 |
|------|---------|---------|------|
| 1 | 首页 | home | 产品入口、资产概览 |
| 2 | 产品列表 | product-list | 金融产品列表 |
| 3 | 产品详情 | product-detail | 产品信息、购买入口 |
| 4 | 交易页 | transaction | 交易操作页 |
| 5 | 账单页 | bill | 交易记录 |
| 6 | 资产页 | asset | 资产管理 |
| 7 | 设置页 | settings | 账户设置 |

## 页面特征

### 首页 (home)

| 特征 | 说明 |
|------|------|
| 核心功能 | 资产概览、产品推荐 |
| 关键模块 | 资产卡片、产品推荐、快捷入口 |
| 用户目标 | 了解资产、发现产品 |

### 交易页 (transaction)

| 特征 | 说明 |
|------|------|
| 核心功能 | 交易操作 |
| 关键模块 | 金额输入、产品选择、确认交易 |
| 用户目标 | 完成交易 |
| 特殊要求 | 安全验证、风控校验 |

### 账单页 (bill)

| 特征 | 说明 |
|------|------|
| 核心功能 | 交易记录查看 |
| 关键模块 | 筛选区、账单列表、账单详情 |
| 用户目标 | 查看交易记录 |
