# 内容类页面类型

## 页面类型列表

| 序号 | 页面类型 | 英文标识 | 说明 |
|------|---------|---------|------|
| 1 | 首页 | home | 内容入口，展示推荐内容 |
| 2 | 内容列表 | content-list | 内容列表、分类列表 |
| 3 | 内容详情 | content-detail | 内容正文、互动区 |
| 4 | 搜索页 | search | 搜索结果页 |
| 5 | 个人中心 | user-center | 用户信息、收藏、历史 |
| 6 | 发布页 | publish | 内容创作页 |
| 7 | 评论页 | comment | 评论列表、回复 |

## 页面特征

### 首页 (home)

| 特征 | 说明 |
|------|------|
| 核心功能 | 展示推荐内容、分类导航 |
| 关键模块 | 搜索栏、分类Tab、内容流、推荐区 |
| 用户目标 | 发现感兴趣的内容 |

### 内容详情 (content-detail)

| 特征 | 说明 |
|------|------|
| 核心功能 | 展示内容正文、互动 |
| 关键模块 | 标题、正文、作者信息、点赞/评论/收藏、推荐 |
| 用户目标 | 阅读内容、互动、分享 |

### 发布页 (publish)

| 特征 | 说明 |
|------|------|
| 核心功能 | 内容创作 |
| 关键模块 | 标题输入、正文编辑、图片/视频上传、标签、发布按钮 |
| 用户目标 | 发布内容 |
