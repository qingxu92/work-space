# 电商类页面类型

## 页面类型列表

| 序号 | 页面类型 | 英文标识 | 说明 |
|------|---------|---------|------|
| 1 | 首页 | home | 平台入口，展示推荐内容 |
| 2 | 商品列表 | product-list | 商品搜索结果、分类列表 |
| 3 | 商品详情 | product-detail | 商品信息、购买入口 |
| 4 | 购物车 | cart | 已选商品管理 |
| 5 | 结算页 | checkout | 订单确认、支付 |
| 6 | 订单列表 | order-list | 订单管理 |
| 7 | 订单详情 | order-detail | 订单信息、物流、售后 |
| 8 | 个人中心 | user-center | 用户信息、订单入口 |
| 9 | 会员中心 | member-center | 会员权益、积分 |
| 10 | 活动页 | activity | 营销活动落地页 |

## 页面特征

### 首页 (home)

| 特征 | 说明 |
|------|------|
| 核心功能 | 展示推荐商品、活动入口、分类导航 |
| 关键模块 | 搜索栏、Banner、金刚区、商品推荐、活动区 |
| 用户目标 | 快速找到想要的商品 |

### 商品详情 (product-detail)

| 特征 | 说明 |
|------|------|
| 核心功能 | 展示商品信息、引导购买 |
| 关键模块 | 商品图片、商品信息、规格选择、评价、推荐 |
| 用户目标 | 了解商品、决定购买 |

### 购物车 (cart)

| 特征 | 说明 |
|------|------|
| 核心功能 | 管理已选商品、引导结算 |
| 关键模块 | 商品列表、数量调整、价格计算、结算按钮 |
| 用户目标 | 确认购买商品、修改数量 |

### 结算页 (checkout)

| 特征 | 说明 |
|------|------|
| 核心功能 | 订单确认、支付 |
| 关键模块 | 收货地址、商品清单、优惠、支付方式、提交订单 |
| 用户目标 | 完成下单支付 |

### 订单列表 (order-list)

| 特征 | 说明 |
|------|------|
| 核心功能 | 查看和管理订单 |
| 关键模块 | 状态Tab、订单卡片、操作按钮 |
| 用户目标 | 查看订单状态、进行后续操作 |
