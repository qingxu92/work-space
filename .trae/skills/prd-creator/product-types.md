# 产品类型配置

## 支持的产品类型

| 序号 | 产品类型 | 英文标识 | 说明 | 典型产品 |
|------|---------|---------|------|---------|
| 1 | 电商类 | ecommerce | 电商平台、商城、O2O等 | 淘宝、京东、美团 |
| 2 | 内容类 | content | 资讯、社区、博客、视频等 | 今日头条、知乎、B站 |
| 3 | 工具类 | tool | 效率工具、管理系统等 | 飞书、钉钉、Notion |
| 4 | 金融类 | finance | 支付、理财、保险等 | 支付宝、蚂蚁财富 |
| 5 | 社交类 | social | IM、社交网络、论坛等 | 微信、微博、豆瓣 |
| 6 | 教育类 | education | 在线教育、学习平台等 | 网易云课堂、得到 |

## 产品类型特征

### 电商类 (ecommerce)

| 特征 | 说明 |
|------|------|
| 核心流程 | 浏览→加购→下单→支付→收货→售后 |
| 关键模块 | 商品、购物车、订单、支付、会员、营销 |
| 用户角色 | 普通用户、会员、商家、运营 |
| 数据特点 | 商品数据、交易数据、用户数据 |

### 内容类 (content)

| 特征 | 说明 |
|------|------|
| 核心流程 | 浏览→互动（点赞/评论/收藏）→分享→创作 |
| 关键模块 | 内容列表、内容详情、搜索、评论、个人中心、创作中心 |
| 用户角色 | 普通用户、创作者、运营 |
| 数据特点 | 内容数据、互动数据、用户数据 |

### 工具类 (tool)

| 特征 | 说明 |
|------|------|
| 核心流程 | 输入→处理→输出→保存/分享 |
| 关键模块 | 功能页、数据管理、设置、协作 |
| 用户角色 | 普通用户、管理员 |
| 数据特点 | 业务数据、配置数据 |

### 金融类 (finance)

| 特征 | 说明 |
|------|------|
| 核心流程 | 浏览→选择→交易→查询→管理 |
| 关键模块 | 产品列表、交易页、资产页、账单、设置 |
| 用户角色 | 普通用户、VIP用户、客服 |
| 数据特点 | 交易数据、资产数据、用户数据 |
| 特殊要求 | 安全性、合规性、风控 |

### 社交类 (social)

| 特征 | 说明 |
|------|------|
| 核心流程 | 发现→关注→互动→分享→私信 |
| 关键模块 | 动态流、个人主页、消息、发布、设置 |
| 用户角色 | 普通用户、认证用户、管理员 |
| 数据特点 | 关系数据、动态数据、消息数据 |

### 教育类 (education)

| 特征 | 说明 |
|------|------|
| 核心流程 | 浏览→选课→学习→作业→考试→证书 |
| 关键模块 | 课程列表、课程详情、学习页、作业、个人中心 |
| 用户角色 | 学员、讲师、管理员 |
| 数据特点 | 课程数据、学习数据、成绩数据 |

## 扩展说明

如需新增产品类型，请：
1. 在本文件添加产品类型定义
2. 在 `page-types/` 目录创建对应页面类型配置
3. 在 `questionnaires/` 目录创建对应问卷
4. 在 `templates/` 目录创建对应模板
