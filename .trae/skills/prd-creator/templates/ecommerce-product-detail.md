## 2. 页面定位与用户角色

### 2.1 页面定位

| 维度 | 说明 |
|------|------|
| 页面名称 | 商品详情页 |
| 核心功能 | 展示商品信息，引导用户购买 |
| 使用场景 | 用户从列表页/搜索页/分享链接进入，了解商品详情并购买 |

### 2.2 目标用户

| 用户角色 | 用户特征 | 使用目标 |
|---------|---------|---------|
| 普通用户 | 未登录/已登录用户 | 浏览商品信息，决定是否购买 |
| 会员用户 | 已开通会员的用户 | 享受会员价，获得更多优惠 |

### 2.3 页面入口

- 从商品列表页点击商品卡片进入
- 从搜索结果页点击商品进入
- 从分享链接直接进入
- 从购物车点击商品进入

## 3. 功能模块详述

### 3.1 商品展示区

**模块定位**：展示商品核心信息，帮助用户了解商品

**商品图片**：

| 字段 | 类型 | 说明 |
|------|------|------|
| 图片列表 | 图片数组 | 商品图片，支持多张轮播 |
| 轮播间隔 | 数字 | 自动轮播间隔，默认3秒 |

**商品信息**：

| 字段名称 | 字段类型 | 是否必填 | 说明 |
|---------|---------|---------|------|
| 商品名称 | 文本 | 是 | 商品标题 |
| 商品价格 | 数字 | 是 | 商品售价 |
| 原价 | 数字 | 否 | 商品原价（用于划线价） |
| 销量 | 数字 | 是 | 累计销量 |
| 库存 | 数字 | 是 | 当前库存 |
| 商品描述 | 文本 | 否 | 商品简介 |

### 3.2 规格选择区

**模块定位**：用户选择商品规格和购买数量

**规格字段**：

| 字段名称 | 字段类型 | 是否必填 | 说明 |
|---------|---------|---------|------|
| 规格名称 | 文本 | 是 | 如：颜色、尺寸 |
| 规格值 | 数组 | 是 | 可选值列表 |
| 库存联动 | 数字 | 是 | 每个规格组合的库存 |
| 价格联动 | 数字 | 是 | 每个规格组合的价格 |

**数量选择**：

| 规则 | 说明 |
|------|------|
| 最小值 | 1 |
| 最大值 | 当前规格库存 |
| 步进值 | 1 |

### 3.3 操作区

**模块定位**：提供购买操作入口

**操作按钮**：

| 按钮 | 触发条件 | 结果 |
|------|---------|------|
| 加入购物车 | 已选规格且库存>0 | 加入购物车，Toast提示成功 |
| 立即购买 | 已选规格且库存>0 | 跳转结算页 |

## 4. 交互设计

### 4.1 正常流程

```
进入页面 → 浏览商品信息 → 选择规格 → 选择数量 → 点击加购/立即购买
```

### 4.2 状态变化

| 状态 | 说明 | 触发条件 |
|------|------|---------|
| 初始状态 | 显示默认规格（如有） | 页面加载完成 |
| 规格已选 | 显示对应库存和价格 | 用户选择规格 |
| 库存不足 | 禁用购买按钮 | 当前规格库存为0 |

### 4.3 联动逻辑

| 联动项 | 触发条件 | 联动结果 |
|--------|---------|---------|
| 价格联动 | 切换规格 | 显示对应规格价格 |
| 库存联动 | 切换规格 | 显示对应规格库存 |
| 按钮状态 | 库存变化 | 库存为0时禁用按钮 |

## 6. 异常处理

| 异常场景 | 处理方式 | 提示文案 |
|---------|---------|---------|
| 商品下架 | 显示提示，推荐相似商品 | "商品已下架" |
| 库存不足 | 禁用购买按钮 | "库存不足" |
| 未登录 | 弹出登录引导 | "请先登录" |
| 网络异常 | 显示重试按钮 | "加载失败，点击重试" |
| 规格不可选 | 灰色禁用 | "暂无库存" |
